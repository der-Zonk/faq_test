question: "If an infantry model that has an 2+ Armour Save (without parry), fights with hand weapon and shield - does it have to use parry and only has a 3+ Asv as the consequence?"
referenced_rules: |
  GW FAQ
---
While the model has to use parry, parry only improves the armour save - therefore, the model retains its 2+ Asv. Note that you cannot use the parry bonus to get to a 2+ Armour Save by any means.
