question: "Does a Lion Cloak improve the models armour save if it is on a chariot? Does Korhil on a Chariot benefit from The Pelt of Charandis if he uses his Chariots armour save?"
referenced_rules: |
  Arcane Journal High Elves
---
Since Lion Cloak is a special rule that is shared with the whole model it works. The Pelt of Charandis is an equipment that only benefits the wearer - Korhil Lionmane, so in this case he is not able to improve the armour save of the Chariot.
