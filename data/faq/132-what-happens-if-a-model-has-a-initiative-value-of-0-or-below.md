question: "What happens if a model has a initiative value of 0 or below (for example via multiple mantle of stone; Chaos Dwarf magic item)?"
referenced_rules: |
  Model Profiles CRB 96 - 97
---
Characteristics cannot go below 0. Models with Initiative 0 attack in Initiative order, after all attacks with initiative 1, including stomp attacks.
